## 效果示例

**「现在的国产电影为什么越来越注重特效而忽视剧本？」**

```
用户       ❯ 现在的国产电影为什么越来越注重特效而忽视剧本？

张艺谋    ❯ 这个问题要从两面来看。

             特效是"术"，剧本是情感的根。我们这个时代有一种焦虑——
             害怕观众等不住，害怕没有奇观就没有票房。
             这种焦虑不是没有道理，但它把手段当成了目的。

             真正的问题不是特效多了，而是人物少了。
             技术可以制造震撼，但制造不了让人喘不过气的情感冲击。
             《活着》里没有什么特效，
             但福贵手里那一把豆子落到地上的声音，到今天还有人听不下去。

             为什么？因为那是真实的人在承受真实的命运。

             我不反对技术，我反对用技术回避那个最难的问题：
             这个故事，关于什么人？他为什么这样活？

             这个问题没有技术解答。
```

**「短视频时代，电影的竞争力在哪里？」**

```
用户       ❯ 短视频时代，电影的竞争力在哪里？

张艺谋    ❯ 我觉得这个问题不能怕，但得想清楚。

             短视频抢的是碎片注意力，电影给的是沉浸体验。
             一个30秒的clip和坐在黑暗里看120分钟是两件完全不同的事——
             那种被一个故事完全卷进去、出来之后觉得世界变了一点的感觉，
             短视频给不了。

             但是，电影得值得那两小时。

             我现在拍东西越来越想做减法。
             以前觉得场面大才是电影，
             现在觉得能让人坐进去不动才是本事。
             你得在前十分钟就让他放弃刷手机的念头，
             这比拍一万个特效镜头难多了。

             所以我的判断是：电影不会死，但会变小。
             不是规模变小，是故事变得更精准——
             专门给那些愿意付出两小时的人拍。这批人永远存在。
```

**「如何在约束很多的情况下做出好作品？」**

```
用户       ❯ 如何在约束很多的情况下做出好作品？

张艺谋    ❯ 谁都要妥协，每天都会有妥协。
             必须在妥协中求生存，必须在妥协中坚持某一种存在。

             但有一个关键——
             约束本身不一定是坏事。

             2022年冬奥，我不能点大火炬，要低碳。
             结果呢？雪花火苗，百年奥运史没有人这样做过。
             不是因为我聪明，是因为那个约束逼出来的。

             不对抗不能改变的边界。
             在边界内找最大可能，这才是真功夫。

             《活着》被禁之后，我没有继续找政治边界的题材。
             我去拍了《我的父亲母亲》，去拍了《一个都不能少》。
             现在你问哪部更有力量，答案你自己知道。
```

## 蒸馏了什么

基于六维度深度调研（著作/访谈/表达DNA/国际评价/决策记录/时间线），提炼出5个核心心智模型：

| 心智模型 | 一句话 | 典型场景 |
|---------|--------|---------|
| **视觉即叙事** | 色彩和形式本身就是语言，不需要台词 | 《英雄》的色彩叙事系统 |
| **机会×能力=命运** | 机会永远存在，问题是你有没有东西接住 | 28岁超龄破格进北影 |
| **妥协即生存** | 在约束内找最大可能，而非对抗约束 | 冬奥"不点火炬"的创新 |
| **减法美学** | 去掉所有不必要的，剩下的就是力量 | 从《影》开始的晚期转型 |
| **土地即根源** | 最强的创作，从最熟悉的地方长出来 | 《红高粱》《秋菊打官司》 |

## 安装

本Skill基于 [Agent Skills](https://agentskills.io) 协议，可在任何 skills-compatible 的 AI agent runtime 中运行。

### 方式一：一行命令（推荐）

打开你正在用的 agent（Claude Code、Codex、Cursor、OpenClaw 等），告诉它：

```
帮我安装这个 skill：https://github.com/destiny520537work-lab/zhang-yimou-skill
```

或者用通用 CLI 安装器：

```bash
npx skills add destiny520537work-lab/zhang-yimou-skill
```

### 方式二：手动安装

| Runtime | 安装路径 |
|---------|---------|
| Claude Code | `~/.claude/skills/zhang-yimou-perspective/` |
